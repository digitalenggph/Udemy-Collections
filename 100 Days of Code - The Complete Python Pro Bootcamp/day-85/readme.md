## Image Watermarking Desktop App

#### Tools to be used:
```pip install pillow```

```pip install tkinter```

#### Requirements
| Requirement  | Must-have    | Extra      |
|--------------|--------------|------------|
| Upload image | Single       | Multiple   |
| Watermark    | Text Only    | Logo       |
| Preview      | Single Image | All Image  |
| Save Image   | Single       | All Image  |
